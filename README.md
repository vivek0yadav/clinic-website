# 🏥 Dr. Sucheta Clinic Website

A modern and responsive clinic website built using **HTML, Tailwind CSS, JavaScript, and PHP**.

---

## 📌 Project Overview

This website is developed for **Dr. Sucheta's Gynaecology & Cosmetology Clinic** to provide online presence and easy access to clinic services, contact details, and patient engagement.

---

## 🚀 Features

* ✅ Responsive Design (Mobile Friendly)
* ✅ Home Page with Overview Sections
* ✅ Services Page
* ✅ About Doctor Page
* ✅ Gallery Page (Image Grid)
* ✅ Contact Page with Form
* ✅ Patient Reviews Section (Dynamic)
* ✅ WhatsApp Floating Button
* ✅ Clean UI using Tailwind CSS

---

## 🛠️ Tech Stack

* HTML5
* Tailwind CSS
* JavaScript
* PHP

## ⚙️ How to Run Locally

1. Install XAMPP
2. Place project in:

   ```
   htdocs/
   ```
3. Start Apache
4. Open browser:

   ```
   http://localhost/clinic-website
   ```
